data\_management.lila package
=============================

This package contains tools for updating and working with datasets hosted on lila.science.

Though not documented here, since they're not formal modules, see the following for examples of using the lila_common module:

* `create_lila_blank_set.py <https://github.com/agentmorris/MegaDetector/blob/main/data_management/lila/create_lila_blank_set.py>`_: a script for downloading a large, diverse set of blank images from LILA
* `create_lila_test_set.py <https://github.com/agentmorris/MegaDetector/blob/main/data_management/lila/create_lila_test_set.py>`_: a script for creating a test set of empty and non-empty images from LILA
* `download_lila_subset.py <https://github.com/agentmorris/MegaDetector/blob/main/data_management/lila/download_lila_subset.py>`_: a script for downloading a specific set of images from LILA, e.g. "all the foxes"
* `get_lila_image_counts.py <https://github.com/agentmorris/MegaDetector/blob/main/data_management/lila/get_lila_image_counts.py>`_: a script for counting the number of images in each LILA camera trap dataset
